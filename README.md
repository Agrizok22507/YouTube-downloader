# YouTube-downloader
Save YouTube videos by search keywords

# Installation
Install python

Install yt-dlp
```bash
pip install yt-dlp
```
Or
```bash
pip3 install yt-dlp
```
Create file for requsts
```bash
touch requsts.txt
```
Or

Remove the '#' symbol from 59 line on file 'youtube_downloader.py':

    search = input("🔍 Your request for search (split by ','): ")
    
    # with open("requests.txt", "r", encoding="utf-8") as f:
    
    #     search = f.read()

Type request (like "python learn,python backend,yt-dlp python")

Start the program
```bash
python youtube_downloader.py
```
Or
```bash
python3 youtube_downloader.py
```
